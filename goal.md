    # Interactive Vector Editor

  ### Functional Requirements

  - Create points in Point Mode by clicking the canvas.
  - Create polygons in Polygon Mode by clicking to add vertices, then finalize with a Complete button.
  - Move points and polygons in Move Mode using drag and drop.
  - Delete points and polygons in Delete Mode by clicking them.
  - Support Undo and Redo for:
      - creation
      - movement
      - deletion

  ### Technical Requirements

  - Include automated tests for core logic such as:
      - shape creation
      - history stack behavior
      - coordinate calculations
  - Document how to run tests in README.md.
  - If AI tools were used, include the prompts in AI_PROMPTS.md.

  ### README Requirements

  Include in the root README.md:

  - Node.js version
  - package manager used
  - install command
  - development run command
  - test command

  A lockfile is required.

  ### Submission Requirements

  Submit either:

  - a compressed project file excluding node_modules and build artifacts, or
  - a public GitHub repository link

  Ensure README.md is in the project root.
